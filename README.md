- (En/dis)able EIS
- Patch MIUI features
- Unlimited Google Photo original backup until Feb. 1st 2022

# MIUI
- Regular MIUI Camera2API vendor properties

# AOSP/LOS

- Gimmick (selfie, square, portrait) AI
